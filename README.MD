# LeveLEO Запуск проєкту
## Локальна розробка
   ```bash
   dotnet run
   ```
2. Відкрийте браузер і перейдіть за адресою http://localhost:5158/scalar, щоб побачити всі ендпойнти проєкту. 
## Docker
1. Створіть образ Docker за допомогою команди:
   ```bash
   docker compose down
   docker compose build --no-cache
   docker compose up
   ```
2. Відкрийте браузер і перейдіть за адресою http://localhost:8080/scalar, щоб побачити всі ендпойнти проєкту.
